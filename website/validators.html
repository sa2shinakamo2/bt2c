Validator Requirements

Everything you need to get started as a validator

Hardware

  • • 4 CPU cores
  • • 8GB RAM
  • • 100GB SSD storage
  • • Stable internet connection

Software

  • • Linux/macOS/Windows
  • • Docker
  • • Node.js 16+
  • • Git

Staking

  • • Minimum stake: 1.0 BT2C
  • • Early validator reward: 1.0 BT2C
  • • Developer node reward: 100 BT2C
  • • Distribution period: 14 days
