  • ‚Ä¢ Developer node (first ever node): 100 BT2C one-time reward
  • ‚Ä¢ All other validator nodes: 1 BT2C for joining during this period
  • ‚Ä¢ Note: Only validator nodes are eligible for distribution period rewards. Standalone wallet holders cannot participate in the initial distribution.
Regular Block Rewards
  • ‚Ä¢ Instant rewards after each block validation

  • ‚Ä¢ Minimum stake: 1 BT2C

Installation Steps

1. Prepare Your Server

sudo apt update && sudo apt upgrade -y
sudo apt install -y curl git build-essential

# Install Docker
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo usermod -aG docker $USER

2. Get the Code

git clone https://github.com/sa2shinakamo2/bt2c.git
cd bt2c
cp env.template .env

3. Configure Your Node

# Edit .env file
NODE_NAME=your-node-name
NODE_PORT=3000
NODE_HOST=0.0.0.0

4. Start Your Node

docker-compose up -d
docker-compose logs -f

5. Register as Validator

curl -X POST http://localhost:3000/api/v1/validator/register \
  -H "Content-Type: application/json" \
  -d '{
    "validatorAddress": "your-wallet-address",
    "stake": "1000000000000000000"
  }'

Monitoring Your Node

Health Checks
