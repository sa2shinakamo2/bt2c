Validator Setup Guide

System Requirements

Hardware

  • • 4 CPU cores
  • • 8GB RAM
  • • 100GB SSD storage
  • • 10Mbps+ internet connection

Software

  • • Ubuntu 20.04 LTS (recommended)
  • • Docker
  • • Git

Rewards Structure

Initial Distribution Period (First 2 Weeks)
  • • Developer node (first ever node): 100 BT2C one-time reward
  • • All other validator nodes: 1 BT2C for joining during this period
Regular Block Rewards
  • • Instant rewards after each block validation
  • • Similar to Bitcoin's reward system
  • • Minimum stake: 1 BT2C

Installation Steps

1. Prepare Your Server

sudo apt update && sudo apt upgrade -y
sudo apt install -y curl git build-essential

# Install Docker
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo usermod -aG docker $USER

2. Get the Code

git clone https://github.com/sa2shinakamo2/bt2c.git
cd bt2c
cp env.template .env

3. Configure Your Node

# Edit .env file
NODE_NAME=your-node-name
NODE_PORT=3000
NODE_HOST=0.0.0.0

4. Start Your Node

docker-compose up -d
docker-compose logs -f

5. Register as Validator

curl -X POST http://localhost:3000/api/v1/validator/register \
  -H "Content-Type: application/json" \
  -d '{
    "validatorAddress": "your-wallet-address",
    "stake": "1000000000000000000"
  }'

Monitoring Your Node

Health Checks

# Check node status
curl http://localhost:3000/api/v1/health

# View validator status
curl http://localhost:3000/api/v1/validator/status

# Check sync status
curl http://localhost:3000/api/v1/sync/status

Troubleshooting

Common Issues

Node Not Syncing

# Restart the node
docker-compose restart

Connection Issues

  • • Check firewall settings
  • • Verify port 3000 is open
  • • Ensure stable internet connection
