Getting Started

BT2C is a proof-of-stake blockchain that allows users to participate in network validation and earn rewards. This guide will help you get started with BT2C.

Prerequisites

  • Basic understanding of blockchain technology
  • Computer with minimum requirements:
    • 4 CPU cores
    • 8GB RAM
    • 100GB SSD storage
  • Stable internet connection

Installation

Follow these steps to install the BT2C node software: 


# Clone the repository
git clone https://github.com/bt2c/node.git

# Enter the directory
cd node

# Install dependencies
npm install

# Start the node
npm start

Becoming a Validator

Requirements

Hardware

  • 4 CPU cores
  • 8GB RAM
  • 100GB SSD storage
  • 10Mbps+ internet connection

Staking

  • Minimum stake: 1 BT2C
  • First node bonus: 100 BT2C
  • Regular rewards: 1 BT2C per node
  • Distribution: Every 2 weeks

Quick Start Guide

1. Set Up Your Environment

# Update and install dependencies
sudo apt update && sudo apt upgrade -y
sudo apt install -y curl git build-essential

# Install Docker
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo usermod -aG docker $USER

2. Get the Code

# Clone repository
git clone https://github.com/sa2shinakamo2/bt2c.git
cd bt2c

# Set up environment
cp env.template .env

3. Configure Your Node

# Edit .env file with your settings
NODE_NAME=your-node-name
NODE_PORT=3000
NODE_HOST=0.0.0.0

4. Start Your Node

# Build and start
docker-compose up -d

# Check logs
docker-compose logs -f

5. Register as Validator

# Register node (replace with your details)
curl -X POST http://localhost:3000/api/v1/validator/register \
  -H "Content-Type: application/json" \
  -d '{
    "validatorAddress": "your-wallet-address",
    "stake": "1000000000000000000"
  }'

Monitoring

Basic Commands

# Check node status
curl http://localhost:3000/api/v1/health

# View validator status
curl http://localhost:3000/api/v1/validator/status

# Check sync status
curl http://localhost:3000/api/v1/sync/status
