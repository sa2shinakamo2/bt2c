bit2coin Whitepaper v1.0

A next-generation digital store of value combining Bitcoin's proven economic model with an energy-efficient Proof of Stake consensus mechanism

Economic Model

Fixed Supply, Predictable Issuance

Maximum Supply

Fixed at 21,000,000 BT2C, ensuring scarcity and value preservation.

Block Rewards

Initial reward of 21.0 BT2C, halving every 4 years (126,144,000 seconds).

Validator System

Secure and Efficient Consensus

Flexible Staking

1.0 BT2C minimum stake required. Flexible staking with no minimum period - stake and unstake anytime. The first validator node receives a one-time 100 BT2C developer reward. Rewards accrue over time to incentivize longer-term participation.

Initial Distribution

14-day distribution period with 1.0 BT2C early validator reward. All rewards during this period are automatically staked. After distribution ends, standard flexible staking rules apply.

Dynamic Rewards

APY varies based on total network stake, individual stake amount, and validator performance (block validation accuracy, uptime, response time, transaction throughput). Short-term staking possible but less profitable.

Unstaking Process

Withdrawal requests enter an exit queue. Processing time varies with network conditions and queue length. Validators continue earning rewards until their unstaking request is processed.

Reputation System

Performance-Based Selection

Reputation System

Performance-based reputation affects validator selection priority, reward multipliers, and exit queue processing speed. Scores persist across staking cycles and are publicly visible.

Network Access

Main website at bt2c.net, API endpoint at api.bt2c.net, and block explorer accessible at bt2c.net/explorer. Rate limited to 100 requests per minute.

Security

Enterprise-Grade Protection

Cryptographic Foundation

2048-bit RSA keys and BIP39 seed phrases with 256-bit entropy.

Network Protection

SSL/TLS encryption and rate limiting at 100 requests per minute.
