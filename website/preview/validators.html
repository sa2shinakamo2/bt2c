How to Become a Validator

Follow these steps to start validating on the BT2C network

1

Create Wallet

Get the minimum required 1 BT2C token to start validating. Join during the first 2 weeks to receive the initial validator reward.
2

Acquire Tokens

Get the minimum required 1 BT2C token to start validating. First node earns 100 BT2C bonus.
3

Setup Node

Install and configure your validator node using our detailed guide. Requires Linux (Ubuntu 20.04 recommended).
4

Start Validating

Begin validating blocks and earn rewards instantly after each block validation, just like Bitcoin.

Validator Rewards

Earn for securing the network

Join the BT2C network as a validator and earn rewards for your contribution to network security.

Initial Distribution Period

First 2 weeks after project launch. Every validator node receives 1 BT2C for joining during this period.

Developer Node Reward

The first ever node (developer node) receives 100 BT2C as a one-time developer reward.

Block Validation Rewards

Earn rewards immediately after each successful block validation, similar to Bitcoin's reward system.

Minimum Stake

Stake at least 1 BT2C to participate in network validation.
